# 2D Collision Simulation

A physics-based simulation of collisions between moving objects in a two-dimensional environment.

## Overview

This project models elastic collisions using conservation laws from classical mechanics. Objects move, interact, and collide realistically according to physical principles.

## Features

- Real-time collision detection
- Elastic collision response
- Multiple moving objects
- Boundary interactions
- Interactive visualization

## Physics Concepts

- Conservation of Momentum
- Conservation of Kinetic Energy
- Elastic Collisions
- Vector Mechanics
- Newton's Laws of Motion

## Technologies Used

- Python
- NumPy
- Pygame
